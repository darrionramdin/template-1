How To Edit This Template

Changing Styles

Changing styles is extremely easy. Simply edit them in the main.css file located in css/main.css

The main color used is a variation of green (#2ecc71). Colors are taken from the Flat UI Color Palette. You can find more colors here: https://flatuicolors.com/palette/defo

The code in the HTML files are commented where edits can be made easily.
